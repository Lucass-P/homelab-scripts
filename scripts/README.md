# Scripts d'automatisation Homelab
   
   Collection de scripts Python et Bash pour l'administration et la surveillance de mon homelab.
   
   ## Scripts disponibles
   
   - **backup_script.sh** - Automatisation des sauvegardes système
   - **check_services.sh** - Vérification de l'état des services
   - **create_users_batch.py** - Création d'utilisateurs en masse (Active Directory)
   - **monitor_system.py** - Surveillance des ressources système
   - **network_scanner.py** - Scanner réseau local
