# Scripts d'automatisation Homelab
   
   Collection de scripts Python et Bash pour l'administration et la surveillance de mon homelab.
   
   **Scripts disponibles :**
   - [`backup_script.sh`](./scripts/backup_script.sh) - Sauvegarde automatisée
   - [`check_services.sh`](./scripts/check_services.sh) - Vérification de l'état des services
   - [`create_users_batch.py`](./scripts/create_users_batch.py) - Création d'utilisateurs en masse (Active Directory)
   - [`monitor_system.py`](./scripts/monitor_system.py) - Surveillance système
   - [`network_scanner.py`](./scripts/network_scanner.py) - Scanner réseau
